# Encodex

Encodex is a simple web tool for learning and practicing text encoding and decoding.  
It helps students and developers understand how characters are represented in different formats.

---

## Features

- HTML Character References (Encode / Decode)
- HTML Hex Encoding (`&#xHH;`)
- HTML Decimal Encoding (`&#DD;`)
- Unicode Escape Format (`\uXXXX`)
- URL Encoding / Decoding
- Automatic encoding detection
- ASCII table with character highlight
- Reverse input/output button
- Copy result button

---

## How to Use

1. Type or paste text into the input box.
2. Select an encoding mode (or let auto-detect choose it).
3. Click **Encode** or **Decode**.
4. Click **Reverse** to swap input and output.
5. Use **Copy** to copy the result.
